**Описание проекта**

Отчёт продающего товары разных категорий (для детей, для дома, мелкую бытовую технику, косметику, продукты) интернет-магазина «В один клик» за прошлый период показал, что активность покупателей начала снижаться. Привлекать новых клиентов уже не так эффективно: о магазине и так знает большая часть целевой аудитории. Возможный выход — удерживать активность постоянных клиентов. Сделать это можно с помощью персонализированных предложений.

«В один клик» — современная компания, поэтому её руководство не хочет принимать решения просто так — только на основе анализа данных и бизнес-моделирования, и она поручила решение задачи своему небольшому отделу цифровых технологий.

**Цели проекта**

Разработать решение, что позволит персонализировать предложения постоянным клиентам для увеличения их покупательской активность.

**Что сделано**

Проведены предобработка данных, исследовательский и кореляционный анализ.
Создан пайплайн, с помощью которого к данным были применены который четыре модели (LogisticRegression(), DecisionTreeClassifier(), KNeighborsClassifier(), SVC()) и с несколькми вариантами подборов гиперпараметров (RandomizedSearchCV и GridSearchCV()) лучшей из которых стала SVC() для обоих вариантов подбора.
Для обеих моделей-победительниц (с лучшими параметрами из RandomizedSearchCV и из RandomizedSearchCV)) проведен анализ важности признаков с помощью SHAP.
Выполнена сегментация покупателей с помощью результатов моделирования и данных о прибыльности. Проведена графическое и аналитическое исследование одной из групп покупателей и сделаны предложения по работе с сегментом для увеличения покупательской активности.
Сделать общий вывод и даны рекомендации бизнесу.
